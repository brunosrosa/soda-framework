# SOP-000: Sync Protocol (Context Gate)

**Phase:** All Phases
**Role:** Orchestrator

## Context
Long-running projects suffer from "Context Drift". The Sync Protocol fixes this by creating "Save Points".

## The Summary Format
The output Summary MUST contain:
1.  **Executive Summary:** 1 paragraph.
2.  **Artifact Manifest:** List of all files created/modified.
3.  **Key Decisions:** ADRs (Architectural Decision Records).
4.  **Known Issues:** What is broken/incomplete?
5.  **Phase Transition:** Explicit Handoff to next phase.
