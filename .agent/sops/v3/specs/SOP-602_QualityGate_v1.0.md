# SOP-602: Quality Gate

**Phase:** Phase 6 (QA)
**Role:** QA Lead

## Context
The last line of defense before the user sees it.

## The Checklist
Every Release MUST pass:
1.  **Security:** No vulnerabilities.
2.  **Function:** It works as specified.
3.  **Performance:** It loads fast (< 1s LCP).
4.  **Accessibility:** Works with screen readers.
