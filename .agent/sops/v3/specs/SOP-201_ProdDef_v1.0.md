# SOP-201: Product Definition (The Map)

**Phase:** Phase 2 (Product)
**Role:** Product Manager

## Context
"An MVP is not a half-baked product; it is a specific tool to learn."

## User Story Mapping
Don't just list tasks. Map the **User Journey**.
- **Activity:** "Manage Account"
- **Task:** "Update Profile"
- **Story:** "As a user, I want to upload a photo..."

## Acceptance Criteria (Gherkin)
Use `Given/When/Then` syntax to make it testable by Agents.
