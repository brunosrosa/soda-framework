# SOP-301: UX Writing & Narrative

**Phase:** Phase 3 (UX/UI)
**Role:** UX Writer

## Context
"Words are design."

## Principles
1.  **Clear:** No jargon.
2.  **Concise:** Cut the fluff.
3.  **Useful:** Help the user move forward.

## Error Messages
Formula: **Problem + Cause + Solution.**
*Bad:* "Error 500."
*Good:* "We couldn't save your profile (Connection lost). Please try again."
