# SOP-302: Design System (The Interface)

**Phase:** Phase 3 (UX/UI)
**Role:** Frontend Specialist

## Context
Consistency builds trust. A Design System is the single source of truth for UI.

## Tokens
- **Colors:** Use semantic names (`text-primary`, `bg-surface`) not descriptive (`text-blue`, `bg-white`).
- **Spacing:** Use a 4px grid (rem based).

## Components
Follow the **Atomic Design** principle:
- Atoms (Button, Icon)
- Molecules (Search Bar)
- Organisms (Header)
- Templates (Dashboard Layout)
